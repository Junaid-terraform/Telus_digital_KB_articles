System Requirements

  • Processor: i7 (Intel)
  • RAM: 16 GB
  • Supported Browser: Google Chrome
  • Recommended OS: Windows & Mac

Prerequisites for Better Tool Performance:

  • Make sure no VPN or Firewall is enabled.
  • Check the internet speed.
  • Check if the Chrome Browser is up to date.
  • Check if the date and time of the PC are correct.

Basic Troubleshooting Steps

If you encounter any issues, follow the steps below to troubleshoot:

  1. Open the assigned task in app.playment.io.
  2. Right-click the URL and select "Inspect" (Shortcut: Ctrl+Shift+I on Windows, Cmd+Option+I on Mac).
  3. Go to the "Applications" tab:
  4. Navigate to the "Storage" section inside the Applications tab and click "Clear site data".
  5. Right-click the reload button and select "Empty cache and hard reload"
  6. Close the current tab and open the same task in a new tab.
